# Spring Boot 미니 프로젝트

## 목적

백엔드 개발자로서 요구 되어지는 역량 중 하나인 spring 그 중에서도 spring boot 학습을 통하여, RESTful한 API를 설계, MVC 패턴 이해, 복잡한 비즈니스 로직등 안정적인 시스템으로 설계하고 운영할 수 있는 역량을 키우기 위함이다.
